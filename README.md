# Alfred-Workflows
A collection of personal automated workflows for [Alfred App](https://www.alfredapp.com/)

## Open With Fluid
Open a URL with [Fluid Browser](https://fluidbrowser.com/)

### Usage
Run either
```
fl https://www.youtube.com
```
or
```
fluid https://www.youtube.com
```
